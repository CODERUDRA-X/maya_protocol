## 🛑 The Problem (The Context)
We live in an age of cheap dopamine. Social media, substances, and distractions act as a "Maya" (Illusion)—trapping the mind in short-term loops. 
**Maya Protocol** is not just a game; it is a **Gamified Dopamine Control Interface** designed to simulate the consequences of your choices in real-time.

## ⚙️ How It Works (Current Build)
The current **Alpha Version** features a high-stakes decision engine:
* **Biometric Simulation:** Tracks `System Integrity` (Health) vs. `Willpower` (Mental Resilience).
* **The Trade-off Engine:** Every choice has a cost. "Hacking" your energy (via substances) crashes your system later.
* **Cyberpunk Visuals:** A dark, immersive UI built for focus.

## 🛠️ Tech Stack
* **Framework:** Flutter (Cross-Platform)
* **Logic:** Dart (Object-Oriented Architecture)
* **State Management:** Provider
* **Windows Native:** C++ Integration for desktop performance
